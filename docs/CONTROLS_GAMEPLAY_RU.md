# Управление: Игровое Поле (Gameplay)

Этот файл описывает управление во время самой игры (пинбол-стол).

## 1) Основные действия

### PSG / Gamepad

| Действие | Кнопка |
|---|---|
| Левый флиппер | `Left Shoulder (L1)` |
| Правый флиппер | `Right Shoulder (R1)` |
| Плунжер (пуск шара) | `Button South (A)` (удержание/отпускание) |
| Толчок влево (Nudge Left) | `D-Pad Left` |
| Толчок вправо (Nudge Right) | `D-Pad Right` |
| Использовать предмет (Use Item) | `Button West (X)` |
| Пауза | `Start` |

### Клавиатура (симулятор)

| Действие | Кнопка |
|---|---|
| Левый флиппер | `A` или `Left Arrow` |
| Правый флиппер | `D` или `Right Arrow` |
| Плунжер | `Space` (удержание/отпускание) |
| Толчок влево (Nudge Left) | `Left Shift` |
| Толчок вправо (Nudge Right) | `Right Shift` |
| Использовать предмет (Use Item) | `Left Ctrl` или `Left Cmd` |
| Пауза | `Esc` |

## 2) Важное поведение

1. Когда подключен контроллер PSG, игра приоритетно читает его бинды.
2. Клавиатурные fallback-клавиши предназначены для симулятора/ПК-теста.
3. Логика меню и логика игрового поля разделены: в геймплее работают флипперы/плунжер/толчки/предмет.
