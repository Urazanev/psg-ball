# Управление: Главное Меню (Main Menu)

Этот файл описывает текущее управление в главном меню и в окне Shop (капсулы).

## 1) Главное меню (Shop закрыт)

Текущий порядок вертикального фокуса при `W/S` (или `D-Pad Up/Down`):
1. `Connect for Daily` (нижняя кнопка)
2. `Shop`
3. `Launch`
4. `Connect` (верхняя правая)

`Submit` нажимает активную кнопку фокуса.

### Кнопки PSG / Gamepad

| Действие | Кнопка |
|---|---|
| Фокус вверх | `D-Pad Up` |
| Фокус вниз | `D-Pad Down` |
| Подтвердить (Submit) | `Button South (A)` |
| Быстрый вход в Shop | `Button West (X)` |

### Клавиатура (симулятор)

| Действие | Кнопка |
|---|---|
| Фокус вверх | `W` или `Up Arrow` |
| Фокус вниз | `S` или `Down Arrow` |
| Подтвердить (Submit) | `Enter` или `Space` |
| Быстрый вход в Shop | `X` |

## 2) Shop (капсулы)

В Shop доступны две капсулы:
- `Buy 3 balls`
- `Buy 7 balls`

Навигация:
1. `Up/Down` переключает между верхним рядом капсул и нижней кнопкой `MAIN MENU`.
2. В верхнем ряду `Left/Right` переключает фокус между `Buy 3` и `Buy 7`.
3. `Submit` нажимает выбранный элемент.
4. `Back` закрывает Shop.

### Кнопки PSG / Gamepad

| Действие | Кнопка |
|---|---|
| Влево по капсулам | `D-Pad Left` |
| Вправо по капсулам | `D-Pad Right` |
| Переключение рядов (верх/низ) | `D-Pad Up / D-Pad Down` |
| Подтвердить | `Button South (A)` |
| Назад (закрыть Shop) | `Button East (B)` |

### Клавиатура (симулятор)

| Действие | Кнопка |
|---|---|
| Влево по капсулам | `A` или `Left Arrow` |
| Вправо по капсулам | `D` или `Right Arrow` |
| Переключение рядов (верх/низ) | `W/S` или `Up/Down Arrow` |
| Подтвердить | `Enter` или `Space` |
| Назад (закрыть Shop) | `Esc` или `Backspace` |

## 3) Визуальный фокус

Чтобы было видно текущий выбор:
- на главном меню активная кнопка слегка увеличивается;
- в Shop выбранная капсула слегка увеличивается;
- фокус в Shop на `MAIN MENU` также визуально увеличивается.
